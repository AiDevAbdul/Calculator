# Project Overview

This project is a simple command-line calculator written in Python. It allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

## Building and Running

To run the calculator, execute the following command in your terminal:

```bash
python calculator.py
```

The script will then prompt you to select an operation and enter two numbers.

## Development Conventions

The code is written in Python and follows basic procedural programming principles. It is organized into functions for each arithmetic operation, with a main loop to handle user input and interaction.
